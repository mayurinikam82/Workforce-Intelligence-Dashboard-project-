# Workforce-Intelligence-Dashboard-project-
**HR Analytics Dashboard**
This project focuses on analyzing employee data to uncover valuable workforce insights and support data-driven HR decisions. Using Power BI, SQL, and Excel, the dashboard visualizes key metrics such as employee demographics, attrition trends, performance, salary distribution, and departmental analysis. The project helps organizations improve employee retention, workforce planning, and overall HR efficiency through interactive and insightful reporting.
SQL – Used for data extraction, querying, filtering, and analyzing large HR datasets efficiently.

Microsoft Excel – Used for data cleaning, preprocessing, validation, and basic exploratory data analysis.

Data Analytics – Applied statistical and analytical techniques to identify trends, patterns, and actionable business insights.

Data Visualization – Transformed complex HR data into clear and meaningful charts, graphs, and KPI reports for better decision-making
